# Misión

## Qué construimos

Un mapa histórico interactivo que muestra qué países han ganado más Copas del Mundo FIFA (1930–2022). El usuario puede explorar los datos en un mapa coroplético mundial y consultar estadísticas detalladas de cada edición.

1. **Mapa coroplético** — cada país coloreado según su número de títulos.
2. **Tooltip interactivo** — al pasar el ratón, muestra país, títulos y años.
3. **Lista de campeones** — ranking completo con medallas y ediciones ganadas.

## Para quién

- Aficionados al fútbol que quieren visualizar la historia de la Copa del Mundo.
- Cualquier persona curiosa sobre qué países dominan el fútbol mundial.
- El autor, como proyecto portfolio y experimento técnico con D3 + React.

## Principios

- **Datos correctos** — la información histórica debe ser verificable y precisa.
- **UX sin fricción** — el mapa debe ser intuitivo: hover para ver datos, clic para detalles.
- **Stack moderno y ligero** — React 19 + D3 + Tailwind v4, sin backend, cero dependencias pesadas.
- **Calidad visual** — diseño distintivo, nada de estética genérica de IA.

## Qué NO es

- No es una base de datos exhaustiva de todos los partidos del mundial.
- No tiene backend ni API — todo es frontend estático con datos embebidos.
- No es una app móvil nativa (aunque es responsive).
