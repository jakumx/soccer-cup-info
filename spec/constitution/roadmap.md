# Roadmap

## Hecho ✅

1. **001 · Mapa mundial interactivo** — mapa coroplético D3 con tooltip, leyenda y lista de campeones.
2. **002 · CI/CD pipeline** — lint + typecheck + code review automático en PRs, deploy a Amplify, tags y releases.

## v1 — Portada (lanzamiento inicial) 🚀

- [x] Mapa coroplético D3 con tooltip
- [x] Leyenda de colores
- [x] Lista de campeones
- [ ] **003 · Historia del Mundial** — sección informativa sobre el origen de la FIFA, cómo se eligen las sedes y evolución del torneo.

## v2 — Explorador por Edición 🔜

- **Selector de año** — al seleccionar un año, el mapa resalta la sede y muestra info del torneo.
- **Sedes en el mapa** — marcadores/pines en los países anfitriones de cada mundial.
- **Árbol de eliminatorias** — bracket visual (16vos → 8vos → 4tos → Semifinal → Final) con el ganador al centro.
- **Goleadores** — tabla de máximos goleadores del torneo seleccionado.

## Backlog 💡

- **Estadísticas por edición** — al hacer clic en un torneo, mostrar datos detallados (grupos, goleadores).
- **Línea de tiempo interactiva** — filtrar el mapa por rango de años (ej: 1930–1970).
- **Modo comparativa** — seleccionar dos países y comparar sus historiales lado a lado.
- **Data 2026** — actualizar con el Mundial 2026 cuando ocurra.
